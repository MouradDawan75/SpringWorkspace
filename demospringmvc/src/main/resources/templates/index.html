Page d'accueil



Thymeleaf est un moteur de templates (moteur de rendu de document) écrit en Java. Il permet d'intégrer du Javascript et CSS (voir n'importe quel format de texte). Il existe d'autres moteurs de vue: JSP (inclus dans Java EE), FreeMaker, Velocity.... Thymeleaf est le moteur le plus récent et est devenu le générateur utilisé par Spring par défaut.

Par défaut, les templates HTML doivent être placées dans le répertoire src/main/resources/templates de votre projet. Les fichiers statiques (non transfomés par Thymeleaf) doivent être: Javascript, CSS, images..) placés dans le répertoire src/main/resources/static

Thymeleaf utilise attributs insérés dans les tags HTML pour générer du contenu dynamique. Le contenu de ces attributs peut être du cimple texte ou une expression qui sera interprétée par le moteur à l'exécution.

Les expressions sont délimitées par:
${..}: pour les expressions à evaluer
*{..}: pour les expressions de selection
#{..}: pour les expressions de messages
@{..}: pour les expressions d'URL
~{..}: pour les expressions de fragments
